# Module 2: AI Agents for Professional Development

## Module Overview

This module teaches learners to build AI agents that automate and enhance every stage of the professional development journey — from resume optimization to interview preparation to long-term career planning. By the end, learners will have built a **Personal Career Coach Agent** that combines all five capabilities into a single, cohesive system.

## Why This Module Matters

The job market is increasingly competitive and AI-assisted. Professionals who can build (not just use) AI tools for career development have a compound advantage: they improve their own careers while demonstrating exactly the kind of AI skills employers want.

## Target Audience
- Job seekers wanting to automate and optimize their search
- Career changers exploring new paths with data-driven analysis
- Professionals building their personal brand and online presence
- Developers who want to build career-tech products

## Prerequisites
- Module 1: Foundations of Agentic AI (completed)
- Basic understanding of the agent loop, tool-use, and memory
- A Claude API key

## Learning Outcomes

By the end of this module, learners will be able to:

1. **Build** a resume analysis agent that extracts skills, identifies gaps, and suggests improvements
2. **Implement** a job matching agent that scores opportunities against a candidate's profile
3. **Create** an interview preparation agent with mock interviews and feedback
4. **Design** a LinkedIn/professional presence optimization agent
5. **Architect** a career path planning agent that models progression scenarios
6. **Integrate** all five agents into a cohesive Personal Career Coach system

## Module Duration
- **Lecture/Concepts:** 2.5 hours
- **Hands-on Labs:** 4 hours
- **Capstone Project:** 3 hours
- **Total:** ~9.5 hours (self-paced) or 2-day intensive (instructor-led)

## Module Structure

### Section 1: Resume Analysis & Optimization Agent (60 min)
- Parsing resumes: structured data extraction from unstructured text
- Skills taxonomy: mapping extracted skills to industry standards
- Gap analysis: comparing skills against target job requirements
- ATS optimization: keyword density, formatting, section structure
- **Lab 1:** Build a Resume Analyzer Agent

### Section 2: Job Search & Matching Agent (45 min)
- Job description parsing and requirement extraction
- Multi-factor matching: skills, experience, location, salary, culture
- Scoring algorithms: weighted matching with confidence levels
- Batch processing: analyze multiple opportunities at once
- **Lab 2:** Build a Job Matching Agent

### Section 3: Interview Preparation Agent (45 min)
- Question generation: behavioral, technical, situational
- Mock interview simulation with real-time feedback
- STAR method coaching (Situation, Task, Action, Result)
- Company research integration for tailored preparation
- **Lab 3:** Build an Interview Coach Agent

### Section 4: Professional Presence Agent (30 min)
- LinkedIn headline and summary optimization
- Content strategy: post topics, frequency, engagement
- Network analysis: identifying high-value connections
- Personal brand consistency across platforms

### Section 5: Career Path Planning Agent (30 min)
- Career trajectory modeling with skill progression
- Industry trend analysis for future-proofing
- Salary benchmarking and negotiation data
- Transition pathway mapping (current role → target role)

### Section 6: Capstone — Personal Career Coach Agent (180 min)
- Integrate all five agent capabilities into one system
- Multi-agent orchestration with shared memory
- User profile persistence across sessions
- Evaluation and testing of the complete system
